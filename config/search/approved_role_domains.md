### Software Engineering

Any role focused on designing, writing, testing, or maintaining software.

Examples (non-exhaustive): backend, frontend, full-stack, mobile, embedded,
ML/AI engineering, platform/infrastructure engineering, DevOps/SRE, data
engineering, QA/test engineering, SDE/SDET, software developer, programmer.

### IT / Systems

Any role focused on IT operations, systems administration, or end-user
technology support.

Examples (non-exhaustive): sysadmin, network administrator, helpdesk/desktop
support, endpoint management, sysops, IT support, systems engineer, cloud
operations.

### Cybersecurity

Any role focused on information security, defensive or offensive security,
or security governance.

Examples (non-exhaustive): security engineer, SOC analyst, application
security, security operations, GRC (governance/risk/compliance) — when
specifically information-security GRC, penetration testing, incident
response, security architect.
